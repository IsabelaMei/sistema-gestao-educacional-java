# Sistema de Gestão Educacional - Java

Projeto desenvolvido utilizando Programação Orientada a Objetos em Java.

## Funcionalidades

- Cadastro de alunos
- Cadastro de professores
- Cadastro de cursos
- Criação de turmas
- Associação de alunos às turmas
- Registro de avaliações
- Menu interativo via console

## Tecnologias utilizadas

- Java
- Programação Orientada a Objetos
- Collections (List)
- Arquitetura em camadas

## Objetivo

Aplicar conceitos de desenvolvimento backend com Java, reforçando boas práticas de organização de código, separação de responsabilidades e modelagem orientada a objetos.
